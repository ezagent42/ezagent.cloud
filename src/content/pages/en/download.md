## Choose Your Way

EZAgent offers multiple ways to get started. Pick what works best for you:

### Desktop App

Full graphical interface experience. Built-in Python engine + FastAPI + React WebView.

- **macOS**: DMG installer
- **Windows**: MSI installer
- **Linux**: AppImage

### CLI Tool

Developer-oriented command line interface. Install via pip, start with one command.

```bash
pip install ezagent
ezagent start
```

Browser opens the web interface automatically.

### Homebrew (macOS)

```bash
brew install ezagent
```

## System Requirements

- Python 3.10+
- 4GB+ RAM recommended
- Supports macOS, Windows, Linux
